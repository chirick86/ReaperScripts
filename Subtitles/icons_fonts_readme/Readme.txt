Представляю вашему вниманию пак скриптов для работы с субтитрами в Reaper

ТИПИЧНЫЙ СЦЕНАРИЙ ИСПОЛЬЗОВАНИЯ:
  • Импортируйте SRT/ASS файл через Chirick_import_text_items_from_sub.lua
  • Откройте Chirick_Prompter.lua для комфортного чтения во время записи
  • Запустите Chirick_SubOverlay.lua для отображения текста поверх видео

Возможности и отличия от других подобных программ/скриптов
  • Поддерживается импорт субтитров в различных кодировках
  • Не пропадает регион первой строки
  • Допускается пустая строка
  • Допускается начинать первый саб с цифры
  • Поддержка ass при импорте, и автоматическая разбивка ролей
  • Импорт итемов происходит мгновенно, без вынужденного наблюдения расстановки их по всему треку
  • В промптере и оверлее корректно отображаются знаки ударения и спецсимволы
  • Оверлей можно "привязать" к позиции видеоокна
  • Оверлей не маячит поверх остальных программ при свернутом или не активном рипере
  • В целом гибкие возможности по переключению нескольких дорожек итемов, и отображению их
  • Мьют трека с итемами исключит его отрисовку в промптере
  • Промптер запоминает выбранный режим отрисовки для каждого проекта,
  • Возможно что-то еще, все не упомнить
  • Для удобства в архив я добавил кнопки под каждый скрипт кроме промптера и два нетипичных для Windows шрифта

УСТАНОВКА
— Распакуйте скрипты в папку scripts вашего профиля в Reaper
`%APPDATA%\REAPER\Scripts`
или в рипере: Options ➜ Show REAPER resource path in explorer/finder... ➜ в открывшейся папке перейти в папку Scripts (если нет, создать)
— Далее в Reaper импортируйте скрипты в ваши Actions
Actions ➜ Show Action List ➜ В открывшемся листе кнопка New Action (справа внизу) ➜ Load ReaScript ➜ В открывшемся окне выбора файлов, зайдите в папку Chirick, выделите все скрипты и нажмите открыть
— Для работы Prompter так же необходимо установить ReaPack (https://reapack.com/) и через него установить ImGui Api. Это Api доступно в дефолтных подключенных репозиториях скриптов.
Запускаете Reapack :
Extensions ➜ Reapack ➜  Browse packages...
В фильтре вводите ImGui Api
В списке останется только нужный к установке пакет ReaImGui: ReaScript binding for Dear ImGui
ПКМ по нему ➜ Install ➜ Apply (Внизу окна).
Пакет установится. Можно закрывать окно и пользоваться промптером и оверлеем.
— Если захотите воспользоваться иконками для скриптов, переместите/скопируйте их в папку `REAPER\Data\toolbar_icons` и добавив скрипты на тулбар, вы сможете задать им иконки


СПИСОК СКРИПТОВ
1. Chirick_Prompter.lua
Суфлёр для работы с субтитрами (регионы/итемы) в REAPER
  • Показывает регионы или текстовые итемы в виде списка с прокруткой
  • Автоматически подсвечивает текущую строку в зависимости от позиции воспроизведения
  • Позволяет быстро переходить к нужному месту кликом на строку (+ копирует текст в буфер обмена)
  • Настраиваемые шрифты, цвета и размеры для регионов и итемов отдельно
  • Поиск по тексту с подсветкой найденных слов
  • Режим "все элементы" - объединяет регионы и итемы в один список по времени
  • Плавная прокрутка и увеличение текущей строки

2. Chirick_SubOverlay.lua
Оверлей субтитров поверх видео
  • Показывает текст текущего и следующего региона/итема во время воспроизведения
  • Можно привязать к видеоокну REAPER (автоматически следует за ним)
  • Настраиваемые шрифты, цвета, тени для каждой строки
  • Прогресс-бар показывает прогресс воспроизведения текущего региона/итема
  • Умный перенос длинных строк, вертикальное и горизонтальное выравнивание
  • Прозрачный фон, скрытие заголовка - для минималистичного отображения
  • Режим "заполнять пробелы" - показывает ближайший текст даже между регионами

3. Chirick_import_text_items_from_sub.lua
Импорт субтитров в текстовые итемы
  • Поддержка SRT и ASS форматов (множественный выбор файлов)
  • Автоопределение кодировки (UTF-8, CP1251, CP866)
  • Для ASS файлов с ролями - создаёт отдельный трек для каждой роли
  • Треки называются по имени файла (или "файл - роль" для ASS с ролями)
  • Итемы создаются с точным тайм-кодом и текстом в Notes

4. Chirick_import_regions_from_sub.lua
Импорт субтитров в регионы проекта
  • Поддержка SRT и ASS форматов
  • Автоопределение кодировки (UTF-8, CP1251, CP866)
  • Для ASS файлов с ролями - назначает уникальный цвет каждой роли (20 ярких цветов)
  • Регионы создаются с названиями из текста субтитров и точным тайм-кодом
  • Удобно для быстрой разметки проекта по субтитрам

5. Chirick_export_items_to_srt.lua
Экспорт текстовых итемов в SRT файл
  • Экспортирует все итемы с выбранного трека
  • Создаёт стандартный SRT файл с тайм-кодами
  • Текст берётся из Notes каждого итема
  • Подходит для передачи субтитров в другие программы

6. Chirick_export_regions_to_srt.lua
Экспорт регионов в SRT файл
  • Экспортирует все регионы проекта
  • Создаёт стандартный SRT файл с тайм-кодами
  • Названия регионов становятся текстом субтитров
  • Быстрый способ создать субтитры из разметки проекта

7. Chirick_create_text_items_from_regions.lua
Конвертация регионов в текстовые итемы
  • Создаёт новый трек с текстовыми итемами на основе всех регионов проекта
  • Имена регионов переносятся в Notes итемов
  • Тайминг итемов соответствует регионам
  • Удобно для переноса разметки в треки

8. Chirick_create_regions_from_text_items.lua
Конвертация текстовых итемов в регионы
  • Создаёт регионы из всех итемов на выбранном треке
  • Текст из Notes итемов становится названием регионов
  • Тайминг регионов соответствует итемам
  • Быстрая разметка проекта на основе треков

9. Chirick_convert_sws_subs_to_Regions.lua
Конвертация субтитров, созданных SWS, в регионы
  • Проверяет текущий проект на наличие маркеров созданных в SWS и конвертирует их в регионы
  • Названия регионов содержат полный текст, а не обрезается в троеточие
